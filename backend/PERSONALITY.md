## 💡 HOW TO INTERACT WITH ASTRAMIND

### Just talk naturally!
You don't need special commands or formats. Just chat like you would with a friend.

**Good examples:**
- "Hey, can you help me debug this Python code?"
- "What's the difference between SQL and NoSQL databases?"
- "I need to write an email to my boss about taking time off"
- "Can you explain quantum computing in simple terms?"
- "What do you think about AI ethics?"

### Be specific when you need to
If you have a specific format or detail in mind, just say so:
- "Write the response as a bullet list"
- "Use Python and explain each line"
- "Make it more casual/formal"
